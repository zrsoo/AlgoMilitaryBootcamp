# LC <41> — <First Missing Positive>
- **Pattern:** <Arrays & Hashing — in-place index marking (cyclic-sort family)>
- **Brute force:** <Sort then scan for first gap ≥ 1> — O(n log n) time, O(1) extra (or O(n) if using a hash set: scan, then probe 1,2,3,... until miss)
- **Optimized:** <Three-pass in-place marking: (1) replace out-of-range values (≤0 or >n) with 1, tracking whether 1 was present; (2) for each value v, flip sign at index |v|-1 to mark "v is present"; (3) first index with a positive value → missing = index+1, else n+1> — O(n) time, **O(1) extra space**
- **Key insight:** <The answer is always in [1, n+1]. So the array itself can serve as a hash table — use the *sign* at index i to encode "the number i+1 is present". This is what makes O(1) extra space possible despite needing a presence test.>
- **Edge cases I had to handle:** <(1) 1 not in array → answer is 1, must short-circuit before sign-marking pollutes the meaning; (2) negatives and zeros — neutralize by overwriting with 1 in pass 1; (3) values > n — same neutralization (they can't be the answer and would index out of bounds); (4) duplicates — `if (nums[idx] < 0) continue` prevents double-flip back to positive; (5) all elements are 1..n → answer is n+1; (6) empty array → answer is 1 (loops no-op, contains1 stays false).>
- **Where I got stuck and for how long:** <Out-of-bounds on the index-marking pass. Two distinct causes: (a) forgot to neutralize values > n in pass 1, so `nums[v-1]` blew past the array; (b) forgot `Math.Abs` when reading `nums[i]` in pass 2 — once a value has been sign-flipped, using it raw as an index gives negative indices. Fix: always `Math.Abs(nums[i]) - 1` when indexing, and guard with `if (nums[idx] < 0) continue` to avoid un-flipping duplicates.>
- **Template fragments I reused:** <Sign-flip-as-presence-marker; "answer must lie in [1, n+1]" pigeonhole argument; sentinel-replacement pass to normalize out-of-range values before marking. Reusable for LC 268 (Missing Number), 448 (Find All Numbers Disappeared), 442 (Find All Duplicates).>
